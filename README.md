# Salado Cafe Premium UI

Welcome to the **Salado Cafe** website repository. This project is a complete front-end overhaul of a local cafe's landing page, transforming a standard HTML/CSS template into an Awwwards-level, interactive, premium dark-themed experience.

## ✨ Features

This website was designed with a focus on luxury aesthetics, smooth animations, and high-end micro-interactions:

- **Premium Dark Theme**: A custom `#121411` deep forest green and lime color palette reflecting a health-conscious, modern cafe brand.
- **Glassmorphism UI**: Frosted glass effects applied to the navigation bar, category sorting cards, and the custom cursor.
- **Advanced Micro-Interactions**:
  - **Custom Interactive Cursor**: A sleek, custom JavaScript cursor that expands with a frosted glass effect when hovering over interactive elements.
  - **Magnetic Buttons**: Complex Figma-inspired physics where primary buttons and their inner text pull towards the user's cursor dynamically.
  - **Text Mask Reveals**: Elegant, staggered text animations that slide up from invisible masks as the user scrolls.
- **3D Parallax Background**: Organic, floating 3D elements (cherry tomatoes, basil leaves) that react to scroll speed with applied CSS depth-of-field (blurring and brightness variation).
- **Interactive Image Gallery Menu**: The official PDF menu was converted into a dynamic image gallery, fully filterable by category using the glassmorphism hero cards.
- **Corporate Footer**: A structured, 4-column professional footer setup with integrated contact forms, typographic sister-brand logos, and utility rows.

## 🛠️ Tech Stack

This project is built purely with vanilla web technologies to ensure lightweight performance and maximum control over custom animations without relying on heavy frameworks:

- **HTML5**: Semantic structure and SVG icon integration (FontAwesome).
- **CSS3**: Extensive use of CSS Variables, CSS Grid/Flexbox, `backdrop-filter` for glassmorphism, and complex keyframe animations (`@keyframes floatOrganic`).
- **Vanilla JavaScript**: Custom logic for intersection observers (scroll reveals), cursor tracking, magnetic physics calculations, and DOM manipulation for the menu filtering system. No external JS libraries were used.

## 🚀 Running Locally

To view the website locally, you can use any standard live server:

1. Clone the repository: `git clone https://github.com/priyanshu5ingh/salado-cafe-premium.git`
2. Open the directory: `cd salado-cafe-premium`
3. Launch `index.html` in your browser, or use an extension like VS Code Live Server.
